Modulus

Found a number of thousands, hundreds, tens and ones

int number = 1234;
int thousands = number / 1000;
NSLog ("@The thousands place of %d is %d.", number, thousands);
int hundreds = (number % 1000) / 100;
NSLog ("@The hundreds place of %d is %d.", number, hundreds);
int tens = ((number % 1000) % 100) / 10;
NSLog (@"The tens place of %d is %d.", number, tens);
int ones = (((number % 1000) % 100) % 10);
NSLog (@"The ones place of %d is %d.", number, ones);

This is another practical use of the modulus operator - to limit the range of a result to a specific set of numbers.

Bytewise shift operation

Make string uppercase

"fifa".bytes.map { |n| n - 32 }.map(&:chr).join

or downcase

"DOWN".bytes.map { |n| n + 32 }.map(&:chr).join
